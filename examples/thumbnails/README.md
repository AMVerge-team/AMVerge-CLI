# Thumbnails

Generate JPEG thumbnails from video clips. Extracts first keyframe,
resizes to 960px wide, saves as progressive JPEG.

---

## Examples

| file | description |
|---|---|
| [01_make_thumbnails.py](01_make_thumbnails.py) | batch thumbnail generation |

---

## Quick Run

```bash
python examples/thumbnails/01_make_thumbnails.py episode.mp4
```
